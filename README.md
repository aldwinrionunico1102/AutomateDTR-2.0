## AutomateDTR 2.0 - _Web-Based Daily Time Record System_

### Project Description

_A web-based daily time record system using the OutSystems low-code platform with workflow automation, drag-and-drop UI, and integrated database._

AutomateDTR 2.0 is a web application developed to streamline the tracking and management of duty hours for working scholars at De La Salle Lipa. The system replaces traditional manual logging with a digital platform that records time-in and time-out entries, automatically calculates daily, weekly, and monthly work hours, and monitors hour deficiencies or excess time. Designed with a responsive and user-friendly interface, the system is accessible to scholars, office heads, and the LAMP Office, enabling efficient monitoring and approval of duty records. Developed using an Agile project management approach, the application was built through iterative development and continuous feedback to ensure a reliable and practical solution for improving administrative efficiency.

### Technologies Used

- OutSystems – Primary platform used to develop the web application
- Visual Development (Low-Code) – Used to design application logic, workflows, and UI components
- Relational Database (OutSystems Built-in Database) – Used for storing user accounts, time logs, and duty records
- Responsive Web Interface – Ensures accessibility across desktop and mobile devices
- Agile Development Approach – Used for iterative development and continuous feedback during the project lifecycle

### Project Status

This project is completed as an academic project, a final project for the course Software Design, but may still receive improvements or refactoring in the future.
